## Alineacion con roadmap

1. Fase 0: Base del producto (quality gates, auth base, i18n, ingesta IGDB, base SEO).
2. Fase 1: Backlog Core (estados de libreria, valoraciones/reviews, busqueda/filtros/orden, perfiles publicos).
3. Fase 2: Social First (amistades, feed de actividad, reacciones, rachas/objetivos).
4. Fase 3: Motor de Productividad (logica de que jugar, riesgo de abandono, horas totales, recordatorios).
5. Fase 4: Growth y Premium (hardening Steam import, recomendaciones premium, worker para jobs pesados, API mobile-ready).
